🩸 Blood Bank Management System (MERN Stack)

A full-stack Blood Bank Management Web Application built using the MERN stack to manage blood donors, organizations, hospitals, and blood inventory efficiently.

Developer: Chaitanya Kulkarni

Deployed Link:-https://blood-bank-webapp-6p4m.onrender.com
✨ Features

User authentication & authorization (JWT)
Role-based dashboards (Donor, Organization, Hospital)
Blood donation and request management
Blood inventory tracking (IN / OUT)
Organization and hospital management
Responsive user interface

🛠 Tech Stack

Frontend
React.js
Redux Toolkit
Axios
Bootstrap / CSS
Backend
Node.js
Express.js
MongoDB (Atlas)
Mongoose
JWT Authentication


This project was developed to gain hands-on experience with the MERN stack, backend-frontend integration, authentication, and real-world application structure.
